# M318-B002 Expectations

Contract ID: `objc3c-governance-planning-hygiene-policy/m318-b002-v1`

`M318-B002` makes sustainable planning hygiene real in the repo's authoring surfaces.

This implementation must:
- define a stable planning-hygiene policy for new work, amendments, and retirement;
- add a human-readable runbook that tells maintainers how to apply the policy;
- update issue templates so new roadmap and conformance issues declare budget impact
  and exception linkage explicitly.

The implemented policy must require:
- one stated validation posture per issue;
- one explicit budget-impact classification per issue;
- an exception record reference whenever budget impact is `requires_exception_record`;
- preference for shared acceptance and runner surfaces over new milestone-local
  scaffolding;
- bounded notes instead of boilerplate sprawl.
